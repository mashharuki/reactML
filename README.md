# reactML
ReactとNode.jsと機械学習を組み合わせたアプリケーションのリポジトリです。

### データをダウンロードする。
node 1-download.js
### バイナリファイルを解析する。
node 2-conv2csv.js

### データ抽出
node 3-split.js

### 機械学習実行
node 4-train.js

### 精度の確認
node 5-test.js

### node-svmのインストール
node install node-svm

### 必要なモジュールをインストールする。
npm i --save express react react-dom superagent
npm i --save-dev webpack babel-core babel-loader
